# 🖥 문제

비디오 대여 프로그램
> 비디오 대여 관리 프로그램


## ⚠️ 형식    
`입력`    
- 비디오 제목, 장르, 대여 여부, 대여자, 대여 날짜(오늘)   
- 1. 추가 | 2. 수정 | 3. 삭제 | 4. 출력 | 5. 종료   

`출력`   
- 해시맵 내역 출력

`조건`   
- Collection 사용 : HashMap<비디오 제목, 비디오>
- MVC 패턴 형식 유지 : Model(Video class) + Ctroller(VideoController Class) + View(Main)
- 제출
